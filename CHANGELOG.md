# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0

Initial release of Claude Coached — an AI cycling coach powered by Claude Code.

- Core coaching system prompt with periodization, workout analysis, and nutrition protocols
- Athlete data templates (`data/athlete.md`, `data/coaching-notes.md`, `data/training-log.md`, `data/training-plan.md`)
- Example athlete profile for reference
- Workout analysis skill (`/analyse`)
- Strava and Google Calendar MCP integration
- Setup guide, architecture docs, and roadmap
